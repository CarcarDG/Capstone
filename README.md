# Recipe and Food Sharing Platform

A full-stack web application for recipe and food sharing, inspired by Douguo Mei Shi.

## Tech Stack

### Backend
- Spring Boot (JDK 21)
- MySQL Database
- RESTful API

### Frontend
- Vue.js (Node.js 22)
- Modern UI Components

## Project Structure

```
recipe-platform/
├── backend/          # Spring Boot API
├── frontend/         # Vue.js Application
└── database/         # SQL Scripts
```

## Features

### User Features
- Authentication (Login/Register)
- Browse recipes and food diaries
- Recipe search and filtering
- Personal recipe management
- Food diary/notes management
- Weekly meal planning
- Social interactions (like, comment, collect)

### Admin Features
- User management
- Recipe moderation
- Category management
- Announcement management
- Data visualization dashboard
